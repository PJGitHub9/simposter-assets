📦 NFO Cleaner
Batch-clean and standardize .nfo files across your media collection with precision, control, and style.
This PySide6-powered desktop app offers genre swapping, selective tag removal, blank line cleanup, and live logging—all wrapped in a sleek, resizable interface.

🚀 Features
- 🧹 Smart NFO Cleaning
- Remove unwanted tags and blank lines
- Swap genres using multi-rule logic
- Preview changes before applying
- 📁 Folder Management
- Add, remove, and reload folders
- Drag-and-drop support
- Recent folders menu with history
- 🧠 Selective Tag Removal
- Scan for all tags in your .nfo files
- Choose exactly which ones to remove
- Use "Select All" and "Deselect All" helpers
- 🎭 Genre Standardization
- Build rules like Sci-Fi → Science Fiction
- Remove genres entirely or replace with custom values
- Apply all rules in one pass
- 📋 Live Logging & Preview
- Real-time log viewer with timestamps
- Dry-run mode to simulate changes
- Detailed log file saved per session
- 🖥️ Responsive GUI
- Resizable panes with layout persistence
- Background threading for smooth performance
- Settings saved automatically (recent folders, layout, etc.)
